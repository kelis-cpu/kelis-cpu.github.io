php手册

1. 如果比较一个数字和字符串或者比较涉及到数字内容的字符串，则字符串会被转化为数值来进行比较。当用===或！==进行比较时则不进行类型转换，此时类型和数值都要进行对比

2.类型转换的判别

PHP在变量定义中不需要明确类型定义变量类型是根据使用该变量的上下文所决定的。

自动类型转换的例子就是乘法运算符*,如果任何一个操作数是float,则所有操作数都被当成float，结果也是float，否则操作数就被解释为integer。

3.字符串转换为数值

当一个字符串被当作一个数值来取值，其结果和类型如下：

如果该字符串没有包含 ‘.’，’e’ 或 ‘E’ 并且其数字值在整型的范围之内（由 PHP_INT_MAX 所定义），该字符串将被当成 integer 来取值。其它所有情况下都被作为 float 来取值。

该字符串的开始部分决定了它的值。如果该字符串以合法的数值开始，则使用该数值。否则其值为 0（零）。合法数值由可选的正负号，后面跟着一个或多个数字（可能有小数点），再跟着可选的指数部分。指数部分由 ‘e’ 或 ‘E’ 后面跟着一个或多个数字构成。

注：php如何将科学计数法转数字：首先通过if语句判断指定的数值是否为科学计数法然后提取科学计数法中有效数据接着正式处理该数据最后调用convert_scientific_number_to_normal方法实现转换。

结合上述三点，可以知道如果找到一个md5哈希前后开头都是0e且哈希前后字符开头0e后面都是纯数字的字符串就可以绕过level 2

import hashlib
s="0e"
for i in range(10**100):
    s="0e"
    s+=str(i)
    m=hashlib.md5(s.encode("utf8")).hexdigest()
    if m[:2]=="0e":
        if m[2:].isdigit():
            print(i)
            break

脚本运行完得到，md5=0e215962017

get_flag

最后一关，看到了system（）,应该是通过命令执行cat，tac等拿到flag，想要进行命令执行，get_flag参数需要先满足前面两个if条件，不能含有空格，不能使用cat,因为会被替换成wctf2020

1.绕过对cat的过滤，上一篇文章里面有，可以采用tac，sort，ca\t等命令

2.bash环境绕过空格

<  -- 重定向，如cat<flag.php
<>  --重定向
${IFS} 
$IFS$9
${IFS},$IFS,$IFS$9的区别，首先$IFS在linux下表示分隔符，单纯的cat$IFS2,bash解释器会把整个IFS2当做变量名，所以导致输不出来结果，然而如果加一个{}就固定了变量名，同理在后面加个$可以起到截断的作用，但是为什么要用$9呢，因为$9只是当前系统shell进程的第九个参数的持有者，它始终为空字符串

先ls看看有哪些文件，

应该在最长那一段文件里，直接tac$IFS$9flllllllllllllaaaaaaaaaaaaaaaaaag,拿到flag
