Nmap

Nmap是一个网络连接端扫描软件，用来扫描网上电脑开放的网络连接端。确定哪些服务运行在哪些连接端，并且推断哪个操作系统计算机运行。

1.nmap简单扫描
nmap <target ip address>
root@kali:~# nmap 10.16.93.136
PORT    STATE SERVICE
21/tcp  open  ftp
25/tcp  open  smtp
80/tcp  open  http
110/tcp open  pop3
135/tcp open  msrpc
139/tcp open  netbios-ssn
445/tcp open  microsoft-ds
888/tcp open  accessbuilder
扫描出开放的端口
2.nmap简单扫描，并对结果返回详细的描述输出
nmap -vv <target ip address>
3.nmap自定义扫描
nmap -p(range)<target ip> //range定义扫描的端口范围
nmap -p20-120 ip/nmap -p80,100 ip
4.nmap ping扫描
nmap -sP <target IP>
5.nmap 路由跟踪
nmap -traceroute <target IP> 
6.nmap扫描一个网段
nmap -sP <network address></CIDR>
7.nmap操作系统类型探测
nmap -O <target IP>
8.nmap万能扫描，包含了1-10000端口ping扫描，操作系统扫描，路由跟踪，服务探测
nmap -A <target IP>
9.nmap从指定文件获取ip扫描结果定向输出
nmap -iL filepath/filename1 -oN/-oX/-oG filename2
nmap -iL /flag -oN 1.txt

详细nmap使用方法

利用nmap进行文件读取

-iL从inputfilename文件中读取扫描的目标。在这个文件中要有一个主机或者网络列表，由空格，制表键，回车键作为分割符。如果使用该参数，nmap就会从标准输入stdin读取主机名字。

-oN把扫描结果重定向到一个可读的文件logfilename中。

nmap -iL /inputfilename -oN logfilename
假如根目录下有一个inputfilename，那么使用上面命令就将读取inputfilename结果写到当前目录下的logfilename中。

由上述测试结果可知，如果在某脚本文件中使用system()等函数来执行命令，可能会造成已知文件读取。
