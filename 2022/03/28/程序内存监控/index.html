# 程序内存监控

# 1.proc - 进程信息伪文件系统

proc 文件系统是一个伪文件系统，用作内核数据结构的接口，通常安装在 /proc ，其中大部分文件都是可读的，有些文件允许修改内核变量。

**/proc/[pid]** 子目录：每个运行的进程都有一个数字子目录，子目录由进程 ID 命名。每个子目录都包含相同文件名的伪文件和目录，在开发内存监控主要用到两个子目录 /proc/[pid]/statm 和 /proc/[pid]/smaps .

# 2. 虚拟内存和内存的关系

两个问题：

  • 我的程序运行时究竟占了多大内存？
  • 我的程序占用的虚拟内存非常大，正常吗？

虚拟内存就是一个逻辑存在的内存空间，在程序运行过程中虚拟内存中需要被访问的部分才会被映射到物理内存中。虚拟内存大只能表示程序运行过程中可访问的空间比较大，不代表物理内存空间占用也大。

** 驻留内存：** 指那些被映射到进程虚拟内存空间的物理内存，进程的驻留内存就是进程实实在在占用的物理内存。一般我们所讲的进程占用了多少内存就是说占用了多少驻留内存而不是多少虚拟内存。
